# 智能康复分析系统 - 项目汇报

## 1. 项目概述

**标题**：智能康复分析系统 - Multi-Agent + RAG 技术应用

**内容**：
- 项目背景：语言/心理障碍儿童康复需求
- 技术特点：结合多智能体和检索增强生成技术
- 应用场景：康复评估、个性化方案生成
- 项目价值：提高康复效率，提供科学依据

**视觉元素**：

## 2. 技术架构

**标题**：技术架构与核心组件

**内容**：
- **前端**：HTML5 + CSS3 + JavaScript
- **后端**：FastAPI + Python
- **RAG系统**：
  - Embedding模型：nomic-embed-text
  - 向量存储：FAISS
  - 知识库：knowledge_base.json
- **多Agent系统**：
  - 症状分析Agent
  - 风险评估Agent
  - 康复方案Agent
  - 调度Agent

**视觉元素**：
- 架构图：前后端 + RAG + 多Agent
- 代码结构目录树

## 5. 应用案例

**标题**：应用案例与效果

**内容**：
1. **案例一：发育性语言障碍**
   - 症状：发音不清、语言表达困难、注意力不集中
   - 分析结果：发育性语言障碍（DLD）
   - 康复方案：3-6个月分阶段训练

2. **案例二：选择性缄默症**
   - 症状：特定场合不说话、社交回避
   - 分析结果：选择性缄默症
   - 康复方案：渐进暴露和认知行为疗法

3. **案例三：口吃**
   - 症状：重复音节、延长发音、说话紧张
   - 分析结果：言语流畅性障碍
   - 康复方案：流畅塑形和口吃修正训练
